Ruby - 方法

發表於2023-12-30|Ruby

方法 Method

可以重複使用。
定義程式碼更容易理解。

定義方法

  • 在 Ruby 定義方法，使用的是 def 這個關鍵字。

名稱：蛇式命名法、命名很重要。
參數：括號 ( )、多個以逗號分開、可有可無。

使用與呼叫方法

  • 在 Ruby 執行呼叫方法，可以省略小括號，為了讓程式碼看起來更像是一般的文章。

參數 Parameter 與 引數 Argument

參數：(someone)
引數："Justin"

參數預設值

參數個數

  • 方法如果 最後一個參數 是一個 雜湊 的話，雜湊 的大括號可省略。

回傳值

  • 回傳 等於交回 控制權。

return 可以 適時 的省略，方法會自動回傳 最後一行 的 執行結果。

變數 與 方法的優先權

問號跟驚嘆號對方法的影響

  • 問號和驚嘆號是方法命名的一部分，但只會放在最後面。

問號通常會回傳 真假值

驚嘆號可能會有 副作用 或 驚喜

  • 陣列方法加上 ! 會破壞原本的陣列。

函數 Function 跟方法 Method 的不同

  • 函數 Function：沒有作用對象、引數放進去、會有結果產生。
  • 方法 Method：有作用對象。
