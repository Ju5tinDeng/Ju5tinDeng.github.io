字串安插

  • 字串可以使用加號來進行與變數的組合、串接。
  • 透過 #{ } Interpolation 填入變數組合、串接。

跳脫字元 (escape) \

如果想要在雙引號裡放雙引號，或是單引號裡放單引號，必須使用跳脫字元。

告知 Ruby：這是一個普通的引號，不是用來包字串的那個引號。
