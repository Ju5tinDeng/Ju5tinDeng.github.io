陣列 Array

一個蘿蔔、一個坑。
就像是藥盒子一樣的東西。
通常使用中括號 [ ] 來定義和操作陣列，裡面的元素不限定只能存放同一種類型。

索引值

  • 陣列裡每個元素的位置都有一個索引值 (Index)。
  • Ruby 的陣列索引值是由 0 開始算的，所以 0 表示是第 1 個元素。
  • 如果從後面開始算，則用負的索引值則是表示。
