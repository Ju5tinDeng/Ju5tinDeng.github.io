常用的終端機指令

發表於2023-12-16|Terminal

終端機 是什麼 ?

終端機 是一台電腦或者電腦系統，用來讓使用者輸入資料，及顯示其計算結果的機器，簡而言之就是人類使用者與計算機互動的裝置。

與一部獨立的電腦不同，但也是電腦組成的部分。

終端機指令 (MacOS)

輸入指令 指令說明
pwd 顯示目前所在的位置
ls 列出目前的檔案列表
mkdir 建立新的目錄
cd 切換目錄
rm -r 刪除目錄
touch 建立檔案
cp 複製檔案
mv 移動檔案 或 改名檔案
rm 刪除檔案
clear 清除終端機內容

快捷鍵

輸入指令 指令說明
Ctrl + L 清空終端機畫面
Ctrl + W 刪除一個 word
Ctrl + U 刪除一整行
Ctrl + R 搜尋模式
Ctrl + C 中斷進行中的程序
