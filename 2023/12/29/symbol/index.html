Ruby - 符號

發表於2023-12-29|Ruby

符號 Symbol

想像它是 有名字的物件。
符號不是變數，它就只是一個 值 或是 物件。

字串跟符號的差別

字串的內容可以變，但符號不行。

  • 字串
  • 符號

可以把符號當做是不可變的字串。

字串的效能稍微差一點點

  • 字串

字串 在 Ruby 記憶體空間的序號，每次都會顯示不同的序號。

冷凍字串 freeze：每次都會顯示相同的序號。

  • 符號

符號 在 Ruby 記憶體空間的序號會是同一個。

同型別的比較：符號比字串快

符號 的處理速度比 字串 快得多，那是因為 符號 在做比較的時候，是直接比對這兩顆物件的 object id 是不是相同，而 字串 比較則是一個字母一個字母逐一比對。
所以在效能上來說，字串 在做比較的時間複雜度會隨著字母的數量而增加，但 符號 的比較因為只比較是不是同一顆物件，所以它的時間複雜度是固定的。
