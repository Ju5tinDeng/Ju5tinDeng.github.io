名詞解釋

發表於2023-12-18|Explanation

DNS

Domain Name System
網域名稱系統。
像是網址翻譯機。
它將人們可讀取的網域名稱 (www.xxxxxx.com) 轉換為機器可讀取的 IP 地址 (123.4.5.67)。

HTTP VS HTTPS

超文字傳輸通訊協定。
HTTPS 是有加密過的通訊協定。
例如：購物網站 (牽涉到金流)。

RVM

Ruby Version Manager
是一個命令行工具，提供 Ruby 版本環境的管理和切換。
就像是 Ruby 的版本控制器。

Gem

RubyGems

Ruby 的一個包管理器，提供了分發 Ruby 程式和函式庫的標準格式「gem」，旨在方便地管理 gem 安裝的工具，以及用於分發 gem 的伺服器。

IRB

Interactive Ruby
IRB 是一個互動的 Ruby 環境，可以讓我們練習和語法，做些簡單的實驗。

REPL

Read-Eval-Print Loop
交互式頂層構件。
是一個簡單的、交互式的編程環境。
可以作爲一個獨立的程序運行，也可以很容易地包含在其他程序中作爲整體程序的一部分使用。

OOP

Object-Oriented Programming
物件導向程式設計。
具有物件概念的程式設計典範，同時也是一種程式開發的抽象方針。
它可能包含資料、特性、程式碼與方法。
物件則指的是類別 (class) 的實例。

它將物件作為程式的基本單元，將程式和資料封裝其中，以提高軟體的重用性、靈活性和擴充性，物件裡的程式可以訪問及經常修改物件相關連的資料。
