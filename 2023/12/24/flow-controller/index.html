流程控制 (Flow Controller)

根據條件運算的結果改變程式執行的順序。
滿足條件時，會執行某一程式區塊，而條件不滿足時，則執行另一個程式區塊。

邏輯判斷

程式中用來判斷條件是否成立的過程。

等於 =

  • 一個等於 = 是 指定給值 的意思
  • 兩個等於 == 是 比較 的意思
  • 三個等於 === 是 比較或是其他 的意思
  • 驚嘆號加等於 != 是 不等於 的意思

或者 ||

  • 用於組合兩個條件，判斷只要一個條件符合即為成立。

而且 &&

  • 用於組合兩個條件，判斷兩個條件都必須符合才有成立。

nil 和 false

  • 在 Ruby 裡因為所有的東西都是物件，只有 nil 跟 false 會被當假的 false。
  • 事實上 nil 跟 false 也都是 物件。

nil 是 NilClass 類別的實體，而 false 則是 FalseClass 類別的實體。
