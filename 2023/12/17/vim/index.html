Vim 是什麼 ?

Vim 是從 vi 發展出來的一個文字編輯器。
其代碼補完、編譯及錯誤跳轉等方便編程的功能特別豐富，在程式設計師中被廣泛使用。

Vim 模式介紹

  • 普通模式 (Normal Mode)
  • 輸入模式 (Insert Mode)

常用指令列表

開啟要編輯的檔案

快捷鍵

輸入指令 指令說明
I 進入輸入模式
V 進入選取模式
ESC 回到普通模式
:Q 關閉這個檔案
:Q! 不存檔並強制關閉檔案
:W 進行存檔
:WQ 存檔並關閉檔案

普通模式 的操作

輸入指令 指令說明
/搜尋內容 搜尋文字
N 跳上一個結果
n 跳下一個結果
:set nu 顯示行數
G 跳到文件底部
gg 跳到文件頂部
{ 向上跳一個段落
} 向下跳一個段落
B 向前跳一個字
W 向後跳一個字
$ 跳到行尾
0 跳到行頭

輸入模式 的操作

輸入指令 指令說明
U 還原
Ctrl + R 下一步
YY 複製一整行
DD 剪下一整行
P 向下貼上
p 向上貼上

選取模式 的操作

輸入指令 指令說明
Y 複製選取文字
D 剪下選取文字
