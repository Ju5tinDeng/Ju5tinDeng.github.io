Ruby - 類別

發表於2024-01-01|Ruby

類別 Class

類別 (Class) 就像夜市雞蛋糕的模具，會做出很多不同形狀的雞蛋糕。
各種形狀的雞蛋糕，會稱它為 類別 (Class) 做出來的 實體 (Instance)。
每個各別的雞蛋糕，實體 (Instance) 也會稱它為 物件 (Object)。

  • 物件 (object)

物件 (object) = 狀態 (state) + 行為 (behavior)
物件 = 狀態 (名詞) + 行為 (動詞)
就是將一塊記憶體位置 擬人化。

定義類別

類別的命名規定：必須是 常數。

使用類別來產生實體

繼承 Inheritance

類似 生物分類表 的 界、門、綱、目、科、屬、種。
把共同的特徵放在同一個 分類 (class)。
與其說是 繼承 ，不如說是 分類。

有一個 小狗 Dog 類別 及一個 小貓 Cat 類別，各自都繼承 動物 Animal 類別，
小狗 Dog 類別 及 小貓 Cat 類別 的實體都會有 走路 walk 及 吃東西 eat 這兩個方法。

初始化 Initialize

  • 當物件被實體化後，可預設事件。
  • 當物件被實體化後，可帶參數和引數。
  • new 方法的引數，會由 initialize 方法收下，參數和引數 個數必須相同。

new 跟 initialize 的關係

new 像創建一個角色。
initializa 像當角色創建完成後做的第一件事。

實體方法 與 類別方法

實體方法 Instance Method

作用在 實體 上的 方法。

類別方法 Class Method

作用在 類別 上的 方法。

定義方法名稱前面加上 self。

實體變數 與 類別變數

實體變數 Instance Variable

存活在每個 獨立 的實體內，在實體裡面可自由取用的變數，但在實體外就不行取用。

實體變數 是 @ 開頭的變數。

取用實體變數

因為 Ruby 並沒有 屬性 這樣的設計，要取用實體變數，需要另外定義的方法才行。

Ruby 可以適時的 省略小括號，方法 看起來像 屬性。

Getter Method

可以改用 attr_reader 的 讀取 方法。

Setter Method

可以改用 attr_writer 的 設定 方法。

同時設定 Getter Method 與 Setter Method

可以改用 attr_accessor 同時能 讀取 + 設定 的方法。

類別變數 Class Variable

在類別方法裡面可取用的變數。

類別變數 是 @@ 開頭的變數。

開放類別

重複定義類別

同時存在兩個同名的類別不會覆蓋而是 融合。

開放類別 Open Class

Ruby 特有
幫現有的類別加功能，甚至內建的類別也做得到。

方法的存取控制

封裝 Encapsulation：有些功能不想讓外部直接存取。
public / private / protected

public

public：預設，所有的類別都可以存取它。

private

private：不能 有明確的 訊息接收者，呼叫方法的時候 不會有小數點。

其實 private 也不是那麼 private。

使用 send 方法

訊息與接收者

訊息：say_hello()
接收者：kitty

protected

protected：不限定 有沒有明確的訊息接收者。

基本上不太需要 protected，private 更接近 保護 的概念。
